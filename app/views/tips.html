Generelle tips

Start med en relasjon uten fremmednøkler.

Sørg for at datatypen til FKen er lik som den attributten den "linker" til.

Create Table

CREATE TABLE X ( y CHAR(4), z VARCHAR2(30), PRIMARY KEY(y), FOREIGN KEY(z) REFERENCES Æ (ø) );

Insert Into

INSERT INTO X VALUES('0000', 'Tekst');

I dette eksempelet er det selvsagt nødvendig at 'Tekst' er en verdi for attributtet ø i Table Æ, hvis ikke vil man få referanseintegritetsfeil, da FKen z ikke refererer til noe som finnes i Table Æ.

Spørringer

SELECT kolonn(er) FROM tabell(er)

SELECT z FROM X;

SELECT z, y FROM X;

SELECT * FROM X;

SELECT z FROM X WHERE y='0000';
